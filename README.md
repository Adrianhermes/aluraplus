# Alura Plus

Landing page estatica inspirada na plataforma Alura+, desenvolvida durante os estudos de HTML e CSS.

# Acesse ele por esse link:
https://adrianhermes.github.io/aluraplus/ - Acesso pelo Git Pages
https://aluraplus-brown-psi.vercel.app/ - Acesso pelo Vercel

## Visao geral

O projeto apresenta uma pagina de assinatura com:

- secao principal com proposta do Combo+
- secoes de beneficios do servico
- lista de dispositivos compativeis
- rodape institucional

## Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts (Inter)
- CSS Grid e Flexbox
- Variaveis CSS para padronizacao de cores

## Como executar localmente

1. Clone este repositorio:

```bash
git clone <URL_DO_SEU_REPOSITORIO>
```

2. Acesse a pasta do projeto:

```bash
cd nome-do-repositorio
```

3. Abra o arquivo `index.html` no navegador.

Se preferir, use a extensao Live Server no VS Code para rodar em ambiente local.
